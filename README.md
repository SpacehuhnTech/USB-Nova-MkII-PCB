# USB-Nova-MkII-PCB

USB Nova MkII Hardware Design Files

USB Nova is a cute little USB “drive” that acts as a keyboard, much like the USB Rubber Ducky, WiFi Duck, and other BadUSBs. 

This is the USB-C version.
